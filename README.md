# Yox

Just like Vue, but it is more lightweight and easy than Vue.

The key is that yox supports IE6.


> QQ 群 232021930，**可用于生产环境，没有 Vue 使用经验不建议使用，谢谢**。

## Install

CDN

```html
<script src="https://unpkg.com/yox@latest"></script>
```

NPM

```
npm install yox
```

YARN

```
yarn add yox
```
